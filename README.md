# EventAttendance
This is a small piece of code to create Outlook event with preset
My manager wants the group members to send out an calendar event when they ask for day off, training, early leave or late to other members with 3 mandatory requirements
- no ersponse required
- Reminder: None
- Status: Free

Other requirements:
- Default recipient: sample@sample.com
- Default subject with Name
- subject can base on parameter: day off, training, early leave or late
